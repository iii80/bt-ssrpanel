基于宝塔面板一键安装ssrpanel脚本。


----------------------------------------------------------------------------
前提条件：

1.安装宝塔面面板，PHP版本选择7.1，其他默认，使用Nginx环境。

2.新建站点，记住域名。

3.新建数据库，记住数据库用户和密码，（数据库名同用户名）。

4.运行下面的脚本：


    wget https://github.com/4kercc/bt-ssrpanel/raw/master/install.sh && bash install.sh


----------------------------------------------------------------------------
根据脚本要求进行操作即可。
